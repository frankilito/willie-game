# 汽船威利：黄金汽笛失窃记

一部 1928 年风格的 3D 开放世界动作冒险网页游戏。黑白橡皮管动画画风，自由攀爬 / 滑翔 / 体力管理 / 系统性交互，支持双人在线联机。

> 基于 1928 年公共领域作品《Steamboat Willie》的独立再创作，与原权利方无隶属或背书关系。
> 角色造型仅取自该片：无手套、豆豆眼/饼切眼、圆鼻、短裤双扣、大鞋。BGM 为公共领域旋律
> 《Turkey in the Straw》与《Steamboat Bill》的 WebAudio 重新编配，未使用原片录音。

## 双人联机

标题界面三入口：

- **单人冒险**：直接进入。
- **创建双人房间**：显示 4 位房间码，发给同伴；对方加入后双方同时进入游戏（你是玩家 1）。
- **加入房间**：输入同伴的 4 位码（你是玩家 2）。

技术：PeerJS + WebRTC P2P，主机权威（怪物/BOSS 由房主模拟）。ICE 配置内置国内可用
STUN（小米/腾讯优先）+ Cloudflare/Google 备选 + openrelay 免费 TURN 兜底。
连不上时按提示尝试同一 WiFi 或手机热点。合作闯关：每人 HP/背包/任务进度独立，怪物与 BOSS 共享。

  黄铜船钟、煤油灯环、信号旗；牲畜农庄（谷仓/风车/弧形篱笆）。

  帆布滑翔翼（开局即得）、落锚下砸（空中左键）、翻滚无敌帧（C）、金手指（2.5s 三连 T = 5 倍移速）。
- **战斗**：4 武器槽耐久制（甲板拖把/锅炉煤铲/锅炉汽笛/鱼叉弓）、舵轮盾格挡减伤 75%、

- **敌人**：发条河鼠/煤球炸弹（引信自爆+殉爆）/铁壳鳄龟/煤烟幽灵/黑猫大副（120hp）共 35 只，
  固定种子布防（联机一致）；**巨型蒸汽鹦鹉号** BOSS（500hp 三阶段：环飞喷燃煤→嘲笑汽笛→
  俯冲震地→靠岸近战窗口；半血锅炉超压；击坠落湖掉黄金汽笛）。
- **剧情**：默片字幕卡过场 → 领拖把 → 清剿汽笛站 → 船钟共鸣仪式（+45 体力/煤铲/心容器）→
  锅炉岛 → BOSS 战 → 拾取黄金汽笛 → 三声回响河道重开 → 结算名单。主线约 5 分钟。
- **音频**：WebAudio 全合成，零音频文件；128 步 8 小节音序器 BGM，攻击与环境机械卡拍；M 静音。

| 左键 | 攻击 · 空中=落锚下砸 · 鱼叉弓按住蓄力 |

| 1/2/3/4 | 切换武器 |

## 调试参数

`?debug=1`（跳过标题开场、全装备、暴露 `window.G/Player/World/Combat/Enemies/Story/Npc/Net`、FPS）、
`&at=shrine|volcano|boss|tower`（传送）、`&q=<quest>`（设任务）、`&t=<时刻>`、`&w=<天气>`、
`&nodraw=1`（跳过渲染只跑逻辑，CI 联机测试用）、`&mp=host|join&room=XXXX`（自动联机）。

## 工程结构

```
index.html          页面骨架（error overlay、缓存穿透 ?v=N）
css/style.css       黑白默片 UI
js/audio.js         WebAudio 合成音效 + 步进音序器 BGM（模块名 Sound）
js/world.js         地形/城镇/生态/天空天气/远景
js/player.js        移动/体力/攀爬/滑翔/翻滚/相机
js/combat.js        武器/格挡/救生圈/鱼叉弓/VFX/投射物
js/enemies.js       5 种小怪 + 蒸汽鹦鹉号 BOSS（dragon 兼容命名）
js/npc.js           7 位 NPC（道具对话开始即发放）+ 金币商店
js/story.js         任务链与过场（say() 一次性标志守护）
js/ui.js            HUD/对话/小地图/胶片叠层/标题菜单
js/net.js           PeerJS 双人联机（主机权威，window.Net 显式挂载）
js/main.js          启动与主循环
vendor/             three r137.5 全局 API + GLTF/DRACO 加载器 + SkeletonUtils + PeerJS
assets/models/      4 个自制骨骼 GLB（tools/gen_models.py 纯 Python 生成，可重跑）
tools/              GLB 生成器
qa/                 playwright-core 测试：soak / net_test / story_test / visual_test
```

## 测试

```bash
cd qa && npm i playwright-core
node soak.js        # 单人 soak：零报错/碰撞体≥40/NPC=7/道具/灰阶检查
node net_test.js    # 双端联机：互见/跟随/35怪一致/伤害路由/BOSS同步（需联网到 PeerJS 信令）
node story_test.js  # 完整剧情链：开场→拖把→汽笛站→船钟→BOSS→汽笛→结算
node visual_test.js # 特效与造型目检截图
```
